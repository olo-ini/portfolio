Building my Portfolio

Code See Live

I built my portfolio using plain HTML, CSS and Javascript.

I wanted it to have a simple design that provided the code, live version and my write up for each project. I had read multiple resources that stressed the importance of showing people your projects first hand. This was an opportunity to do that

I got to work going through HTML, CSS and Javascript learning material. This project allowed me to learn a bit more about how to use these tools. I was also interested in mobile first responsive design. After weeks of general research and practice, with lots of help from the internet and Youtube, I came up with this simple, highly responsive design as my portfolio.

Feel free to test out it's responsiveness as well as check out the code on my github. Follow the link above
